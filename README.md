#奥莱优品商城
  一个基于前端使用uni-app，Vue2，uView，后端Node.js，Express框架开发的购物商城，适配app端和微信小程序端

##后端文件
  /`server` 是后端文件

##启动项目
  使用 `yarn`下载依赖

###后端
  将server在终端开启<br> 
  使用 `yarn`下载依赖<br> 
  使用 `node start`启动项目，默认端口是`3000`

###前端
  在/`utils`/`request` 更换本地主机Ip设置端口`3000`<br>
